Scribus and Cygwin

Thanks to the KDE-Cygwin team, it is possible to run Scribus on Win2k or XP, via Cygwin. Note that this is a much older version of Scribus and is now deprecated as a native port is under way.

The most up to date info can be found on the Scribus Wiki.
