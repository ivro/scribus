Community Resources

The Scribus mailing list is on-line at http://nashi.altmuehlnet.de/mailman/listinfo/scribus. There is both an English and German language interface to the mailing list manager. This is for both end-user support and development issues.

The Bug Tracker is located at http://bugs.scribus.net/

An IRC channel #scribus on freenode.net is quite active. There is usually at least one person from the core development team present there.

There are French and German Scribus documentation sites. French and German versions of this website are making progress and will be viewable at this address.

There is a Polish translation of the "Get Started with Scribus" tutorial at http://linux.hanski.info/static/scribus/index.html
