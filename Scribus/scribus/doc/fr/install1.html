Comment obtenir la source de Scribus

Vous pouvez obtenir la version la plus récente de Scribus à partir de:

  • Site principal de Scribus - http://www.scribus.net
  • Téléchargements Scribus de Sourceforge:

Il faut aussi remercier Anduin.net, un accès SVN anonyme mis à jour quotidiennement : recommandé seulement pour ceux qui veulent suivre les derniers développements. Instructions pour obtenir la source la plus récente via SVN :

Scribus 1.3.3.x - Stable

svn co svn://scribus.info/Scribus/branches/Version13x

Scribus 1.3.4 - Développement

svn co svn://scribus.info/Scribus/branches/Version13x/Scribus

Scribus 1.3.5cvs - Développement - Qt4 Port

svn co svn://scribus.info/Scribus/trunk

N'importe lequel de ces choix téléchargera les sources dans votre répertoire courant. Ensuite :

cd Scribus
make -f Makefile.cvs

ou en option, utilisez cmake. Cmake est obligatoire pour for 1.3.5svn, parce ce que les «autotools» sont enlever aprés 1.3.4

À partir d'ici, suivez les instructions de Compilation et installation.
