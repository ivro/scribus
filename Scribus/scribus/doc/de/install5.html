Scribus 1.3.5svn+ unter Mac OS X kompilieren und installieren

Dieses Dokument beschreibt, wie man Scribus 1.3.5 oder höher aus den SVN-Quellen unter Mac OS X installiert. Momentan enthält es (noch) keine Informationen darüber, wie man ein Installationspaket baut.

Was Sie benötigen:

  • Den MacPorts-Installer (bis zur Version 1.5 darwinports)
  • Qt 4.3.x von Trolltech
  • XCode von Apple, entweder von der OS-X-Installations-CD oder von http://www.apple.com.
  • Scribus-SVN-Quellen

Kompilieren und installieren

Die notwendigen Schritte, um 1.3.5 unter OS X/Aqua zu installieren:

  • Installieren Sie X11 (von den Installations-CDs für OS X oder von der Apple-Entwickler-Website).
  • Installieren Sie XCode. ()
  • Installieren Sie MacPorts (auf www.macports.org erhältlich).
  • Installieren Qt4 4.3.0+ für OSX (auf www.trolltech.com erhältlich).

Installieren Sie einige portierte Programme von MacPorts (alle Abhängigkeiten werden automatisch aufgelöst).

  • Installieren Sie CMake: sudo port install cmake.
  • Installieren Sie freetype2: sudo port install freetype. Die Version, die in OSX 10.4.9 enthalten ist, ist nicht so gut wie diejenige von MacPorts.
  • Installieren Sie lcms: (sudo port install lcms. Dadurch werden auch libjpeg, zlib und libtiff installiert.
  • Installieren Sie cairo: (sudo port install cairo. Dadurch wird auch fontconfig installiert.
  • Installieren Sie libxml2: (sudo port install libxml2.
  • Installieren Sie subversion: (sudo port install subversion ODER installieren Sie subversion aus einer anderen Quelle
  • Optional (empfohlen):sudo port install ghostscript
  • Erstellen Sie ein Verzeichnis, in das Sie die Quellen herunterladen wollen, zum Beispiel in einem Terminal: mkdir -p ~/scribus/135)
  • Wechseln Sie in dieses Verzeichnis: cd ~/scribus/135

Holen Sie sich die Quellen aus SVN:

  • svn co svn://scribus.info/Scribus/trunk/
  • Erstellen Sie ein Verzeichnis zum Kompilieren: mkdir builddir
  • Wechseln Sie in das neue Verzeichnis: cd builddir
  • Rufen Sie den CMake-Befehl auf: cmake -DBUILD_OSX_BUNDLE=1 -DWANT_CAIRO=1 -DCMAKE_INSTALL_PREFIX:PATH=/Users/username/Applications/Scribus.App/Contents/ ../trunk/Scribus/
  • make
  • make install

Wenn alles gutgegangen ist, können Sie Scribus 1.3.5+ nun unter OS X benutzen.
