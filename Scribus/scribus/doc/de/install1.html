Wie Sie die Quelldateien von Scribus bekommen

Die aktuellste Version von Scribus bekommen Sie von:

  • der Scribus-Hauptseite - http://www.scribus.net
  • als Download von Scribus Sourceforge:

Dank N@twork gibt es einen anonymen Zugang zu CVS, wo Dateien täglich aktualisiert werden; empfehlenswert eher für jene, die stets auf dem neuesten Stand bleiben wollen. Und so bekommen Sie die neuesten Quelldateien vom CVS-Server: 

export CVSROOT=:pserver:anoncvs@scribus.info:/cvs

cvs login

cvs -z3 co Scribus

Damit beginnt das Herunterladen der Quelldateien, die in Ihrem aktuellen Arbeitsordner landen. Anschließend:

cd Scribus
make -f Makefile.cvs

Wenn Sie hier angelangt sind, folgen sie bitte den Anweisungen unter Compiling and Installing.
